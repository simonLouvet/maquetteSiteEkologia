Prototype Site Web Ekologia
===========================

Dans le dossier public_html, sont présents les quatre fichiers suivants :

- model_deco.html   : Ce fichier correspond au modèle d'une page, avec un corps vide lorsque l'utilisateur est déconnecté
- model_co.html     : Ce fichier correspond au modèle d'une page, avec un corps vide lorsque l'utilisateur est connecté
- index.html        : Ce fichier correspond à la page d'accueil
- registration.html : Ce fichier correspond à la page d'inscription et de profil d'un utilisateur. Ici, toutes les parties sont affichées, quel que soit le type d'utilisateur.

Pour avoir un aperçu d'un fichier, double-cliquez sur ce dernier. Il s'ouvrira dans votre navigateur par défaut.

Les styles actuels sont des styles par défauts.
Ils seront modifiés prochainement pour obtenir l'aspect final souhaité.
